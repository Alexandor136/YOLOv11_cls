# Обучение модели YOLOv11

## О проекте

В этом репозитории реализован механизм распознавания внутреннего шкафа холодильного оборудования в тестовом варианте. Данная реализация является частью более масштабного проекта, направленного на повышение эффективности сборки холодильного оборудования.

## Используемая модель

- Модель для обучения: `yolo11l-cls`

## Подготовка данных

Данные для обучения и тестирования были подготовлены с помощью скрипта `preparing_dataset.py`, который преобразует исходные данные в необходимый формат.

### Структура датасета

train
    1 calss
    2 calss
    .......
    .......
    .......
---------------------
test 
    1 calss
    2 calss
    .......
    .......
    .......

Каждая папка соответствует отдельному классу с изображениями для обучения или тестирования.

## Обучение и тестирование

- Файл `training.ipynb` содержит код для обучения модели и проведения тестирования.

---

Данный репозиторий включает все необходимые скрипты и инструкции для подготовки данных, обучения и оценки модели YOLOv11.
